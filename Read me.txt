This code is the course project code of embedded system introduction.

This program can control a car with embedded system:
1.control the car speed from 0-100;
2.use '+' and '-' to speed up and slow down;
3.type 'x', 'b' and 'y' to do line-tracking, obstacle avoidance and circular trajectory;
4.type 'w', 'a', 's' and 'd' to go forward, go backward, turn left and turn right;
5.the buletooth module will be used to let the mobile phone to control the car.

The main function and motor control function can be seen in files named 'main.c' and 'MotorControl.c'.
